# ✨ Magic Link Experience

A premium, cinematic web experience for creating and sharing personalized "Magic Links". Perfect for birthdays, surprises, or special messages.

## 🚀 Features
- **Cinematic Storytelling:** Multiple animated scenes using Framer Motion.
- **Personalized Reveal:** Name reveal with magical gold gradients.
- **Interactive Magic:** Custom particle effects and confetti that respond to clicks/taps.
- **Seamless Sharing:** Generates unique links stored securely in Supabase.
- **Mobile Optimized:** Designed to look stunning on both desktop and mobile devices.
- **Audio Experience:** Integrated background music for a complete emotional experience.

## 🛠️ Tech Stack
- **Frontend:** [Next.js](https://nextjs.org/) (React), [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [Three.js](https://threejs.org/)
- **Backend:** [Supabase](https://supabase.com/) (PostgreSQL & Auth)
- **Icons:** [Lucide React](https://lucide.dev/)

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Khangulamgousamjat/Birthday-wisher-.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env.local` file and add your Supabase credentials:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Deploy**
   Built for easy deployment on [Vercel](https://vercel.com).
